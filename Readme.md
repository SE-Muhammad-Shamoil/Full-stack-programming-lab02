## Project Description

A dynamic, interactive quiz application built with HTML, CSS, and JavaScript. This project satisfies the Lab 02 requirements while implementing a modern, premium user interface with glassmorphism and smooth animations.

## functional Requirements Met

- **Quiz Content:** Includes multiple technical questions.
- **Data Structure:** Questions and answers stored in variables.
- **Logic:** Function-based answer checking and score calculation.
- **DOM Manipulation:** Results and feedback are displayed dynamically.
- **Controls:** Features a "Reset Quiz" button to clear state.
- **Conditional Logic:** Provides personalized feedback based on the final score.

## How to Run

1. Open the folder in VS Code or any code editor.
2. Locate `index.html`.
3. Open `index.html` in your preferred web browser (or use Live Server).
